# Still Point Fencing — 2025-26 Season in Review

A season report for Still Point Fencing members and families: medals, most active and most medaled
competitors, the Tucson Open flagship, tournaments hosted, weapon and rating distribution, and
difficulty-weighted points leaders.

**View it:** https://jfowlersiai.github.io/Still-Point-Fencing-26/

These numbers cover Arizona competitions only; many Still Point fencers also compete out of state and
nationally, and those results are not counted here. Full names are shown, as the results are already
public on askFRED. Points use the same difficulty-weighted method as the Arizona Division report.
